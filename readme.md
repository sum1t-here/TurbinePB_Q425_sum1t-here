## 🚀 Quick Start

### Rust Programs

```bash
# Hello World
cd HelloWorld && rustc main.rs && ./main 

# Counter
cd RustCounter && cargo run

# RNG Game
cd RngGame && cargo run
```

### Solana Anchor Program

```bash
cd SolanaCounter
anchor build
anchor test
anchor deploy
```
